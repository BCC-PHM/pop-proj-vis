# ONS Population Projection Visualisation

Using data from the [ONS](https://www.ons.gov.uk/peoplepopulationandcommunity/populationandmigration/populationprojections/datasets/localauthoritiesinenglandz1) to plot projected populations at ages 60, 65, and 70 for local authorities:
- Birmingham
- Solihull
- Dudley
- Sandwell
- Walsall
- Wolverhampton
